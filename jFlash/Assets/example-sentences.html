Example Sentences 1.1 (18MB)

We heard you loud and clear. Since the first release of Japanese Flash, more than anything else, users have asked for example sentences. Now, we've added 50,000 of them - integrated into Japanese Flash's existing study flow - and we think you'll find it Awesome.

First, press "Start" below to begin the download & installation (it's free, of course!). Make sure you have a good network connection (preferably WiFi).

Here's how it works: when you're studying cards, swipe sideways to reveal example sentences. Not every card has example sentences linked to it, but many common words have many to read through.

We've also indexed all of the key words in these example sentences to be searchable. After installing this plugin, you will be able to find example sentences as well using Search.

You may want to know why we're asking you to download this plugin separately from the application itself. That's a great question. And, since you are Awesome (you purchased Japanese Flash after all!), we'll share Our Evil Master Plan.

We separated the components (dictionary search, example sentences, et cetera) into plugins to make it easier to update the data as it changes. We all know language changes - so your iPhone study application needs to change with it. So, if we receive reports of errors in the dictionary, we will be able to roll out changes to everyone automatically, without re-installing the application. Cool, right? That platform isn't quite ready yet - but we are working on it - and this change was a key step along the way.

Enjoy example sentences!
