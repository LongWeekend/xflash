Version 1.1 (7.5MB)

We heard you loud and clear. Since the first release of Japanese Flash, more than anything else, users have asked for example sentences. We've added 50,000 of them - integrated into Japanese Flash's existing study flow - and we think you'll find it Awesome.

Press "start" below to begin the download and installation (it's free, of course!). Make sure you have a good network connection, preferably WiFi or 3G.

Here's how it works

When you're studying cards, swipe sideways to reveal example sentences. Not every card has example sentences, but many over 17,000 words have examples to read through.

We've also indexed all of the key words in these example sentences to be searchable. After installing this update, you will be able to find example sentences as well using Search.

Why do I have to download something?

You may be wondering why we're asking you to download this update separately from the application itself. That's a great question! Since you are Awesome (after all, you purchased Japanese Flash), we'll share our master plan.

We separated the components (dictionary search, example sentences, etc) into updates to make it easier to deliver new data as it changes.

We all know language changes - so your iPhone study application needs to change with it. So if we receive reports of errors in the dictionary, we will be able to roll out changes to everyone automatically, without re-installing the application. Cool, right?

That platform isn't quite ready yet - but we are working on it - and this change was a key step along the way.

Enjoy studying with example sentences!
