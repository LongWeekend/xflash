Click on the 'Study Sets' tab and choose a set by touching its name. Sets are organized into folders and are also searchable.

To search, slide your finger down the screen to reveal a search bar. Search results are updated in real time, and they are fast!

Tap on the 'Study Sets' tab and press '+' to add a set. You can add words to the set using the Word Search. You can also add cards to a set whilst in practice mode using the 'Add To Set' button.

We believe an iPhone is a great way to practice, but not great for creating flash cards. So in version one, this functionality is limited.

You'll be pleased to hear that our roadmap includes some pretty special solutions to this problem.

back

Once a set is chosen, you can either browse cards or test yourself in practice mode. Card meanings are hidden until you touch the card area. Once revealed, you can mark a card as 'Right', 'Wrong', or 'Know It'. You can also add it to other sets with the 'Add To Set' button.

The first time you get a card right, it moves from 'Studying' to 'Right 1x'. Each time you get it right, it will be promoted to a higher level.

Get it right five times and it will be marked as 'Learned'. The progress pop-up shows detailed results for the current set. Tap the progress bar to display it.

Your goal is to get each card right 5 times.
Good luck!

If you feel like some mental warm-ups before practicing, tap the 'Settings' tab and flick the 'Browse Mode' on/off switch. Now you are in browse mode, so you can move backwards & forwards through the set by swiping or tapping next/last.

JFlash has well over 145,000 words separated into more than 100 sets. Word search can return over 200 results in under 1 second using English, Romaji or Japanese keywords.

Search works best if you look for whole words and use the plain or infinitive form for verbs. You can also use multiple search terms at once in either language.

Try using a question mark at the end of a word as a 'wild card'. So searching for 'japan?' would also return results like Japan, Japanese, Japanofile, Japanasaurus Rex, etc...

During beta, we noticed how many people would "borrow" our iPhones to use JFlash! Problem was, all our progress data was mixed up! So we added multiple users.

From the 'Settings' tab press 'Users'. You can select a user by tapping the name. You can add new ones using the '+' button, and you can even select an avatar picture from the iPhone photo gallery for each user. We've added some users to get you started!
